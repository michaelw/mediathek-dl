mediathek-dl
============

mediathek-dl prefetches contents of ARD Mediathek RSS feeds, e.g.,
http://www.ardmediathek.de/export/rss/id=1458


Required Software
=================
- bash
- curl
- rtmpdump
- libxml-xpath-perl
- tatort-dl (included)


Included Software
=================
- tatort-dl was retrieved from:
  http://rg42.org/wiki/tatort-dl
